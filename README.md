# Telechat
เป็น app ไว้สำหรับแชทกันทั้งแบบส่วนร่วมและส่วนตัวมีฟังก์ชั่นมากมายให้ได้ใช้งาน
### ⚙️ How to Run
1. Run Server
    ```java
    java -jar Server.jar
    ```
2. Run Client
    ```java
    java -jar Client.jar
    ```
3. Ready to use
